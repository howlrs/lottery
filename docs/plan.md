## 機能

### GUI部
- APIから円形ボードの要素を取得する
- 円形のボードを回転（A
- Aをタップ及びクリックすると停止する
- 停止位置により、報酬を獲得する
- 獲得した報酬を表示し、APIに対して獲得報酬を送信する

### 管理画面部
- 提供可能報酬を設定（名称、説明、価値、個数）
- 提供可能報酬をリスト化（項目それぞれの当選確率）
- 提供済み報酬を把握管理


### API
- 提供可能報酬リストを取得
- 提供可能報酬を設定
- 提供可能報酬を更新
- 提供可能報酬を削除


## 実装方針
- Nextjsでフロントバックエンドを構築する
- Google Cloud Run, AWS AppRunnerで展開予定
